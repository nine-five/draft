# Nine-Five design
Browse the demo at [here](http://nine-five.github.io/draft).

## Node.js user
1. Install Dependencies `npm install`
2. Run the Application `npm start`
3. Now browse to the demo at `http://localhost:8080/index.htm`.

## Rebuild the css
$ npm install -g stylus
$ cd app/css
$ stylus kanban.styl
